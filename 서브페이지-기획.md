# Nexora AI 랜딩 - 서브페이지 3D 애니메이션 기획

> 포트폴리오 + 추후 개인 웹사이트 재활용 목적

---

## 현재 상태

- **메인 페이지**: 3D 파티클 구체 (스크롤 시 분산) + 5개 섹션 완성
- **추가 목표**: 서브페이지 2개, 각각 다른 3D 애니메이션 적용

---

## 서브페이지 후보 아이디어 (6개)

### 아이디어 A. Technology 페이지 - Morphing Noise Sphere

**컨셉**: AI의 "사고하는 두뇌"를 유기적으로 변형되는 3D 블롭으로 표현

**3D 효과**:
- Perlin/Simplex 노이즈로 구체의 버텍스가 실시간 변형
- 마우스 위치에 반응해 형태가 일그러짐
- 변형 정도에 따라 블루 → 시안 → 퍼플 색상 변화
- 스크롤 시 블롭이 분열/합체하는 효과

**기술 핵심**:
- `ShaderMaterial` 커스텀 vertex/fragment shader
- GLSL 노이즈 함수로 버텍스 디스플레이스먼트
- Cosine palette 함수로 색상 매핑
- `UnrealBloomPass` 글로우 후처리

**난이도**: ★★★☆☆ (중)
**성능**: 가벼움 - 단일 메시 + 셰이더. 모바일 60fps 가능
**와우팩터**: ★★★★☆

**페이지 구성**:
1. 히어로: 거대한 모핑 블롭 + "Our Technology" 텍스트
2. AI Engine 설명 (블롭이 배경에서 계속 변형)
3. 기술 스택 소개 (Python, TensorFlow 등)
4. 벤치마크/성능 수치

---

### 아이디어 B. About 페이지 - Interactive 3D Globe

**컨셉**: Nexora의 글로벌 AI 인프라를 인터랙티브 지구본으로 시각화

**3D 효과**:
- 3D 지구본 위에 데이터 포인트(도시)가 빛나는 도트로 표시
- 도시 간 연결선(arc)이 곡선으로 애니메이션
- 자동 회전 + 마우스 드래그로 직접 조작
- 호버 시 도시명/데이터 팝업

**기술 핵심**:
- `SphereGeometry` 3개 레이어 (바다, 도트, 아크)
- Sunflower Spiral 패턴으로 균일한 도트 분포
- `CubicBezierCurve3`로 아크 경로 계산
- D3.js `geoInterpolate`로 대원 경로 따라 애니메이션

**난이도**: ★★★★☆ (중상)
**성능**: 도트 수 조절로 모바일 대응 가능 (Stripe도 이 방식으로 60fps 달성)
**와우팩터**: ★★★★★

**페이지 구성**:
1. 히어로: 전체 화면 인터랙티브 지구본
2. 글로벌 파트너/고객사 (지구본에 하이라이트)
3. 팀 소개
4. 회사 연혁 타임라인

---

### 아이디어 C. Demo 페이지 - GPGPU Particle System

**컨셉**: AI가 데이터를 처리하는 과정을 수만 개 파티클의 형태 변환으로 시각화

**3D 효과**:
- 파티클이 텍스트/로고 형태 → 구체 → 흩어짐 → 다시 모임 (모핑)
- 마우스 근처 파티클이 밀려남 (repulsion)
- 스크롤에 따라 형태 전환
- 10만+ 파티클도 60fps

**기술 핵심**:
- `GPUComputationRenderer` (Three.js examples)
- Position/Velocity 데이터 텍스처 ping-pong 업데이트
- Fragment shader에서 속도 기반 알파 모듈레이션
- `MeshSurfaceSampler`로 3D 모델 표면에서 파티클 좌표 추출

**난이도**: ★★★★★ (상)
**성능**: GPU 병렬 처리로 10만+ 파티클 가능. 모바일은 파티클 수 축소 필요
**와우팩터**: ★★★★★

**페이지 구성**:
1. 히어로: "NEXORA" 텍스트가 파티클로 구성 → 스크롤 시 구체로 모핑
2. 데모 인터랙션 영역 (마우스로 파티클 밀어내기)
3. 제품 데모 시나리오 (스크롤 기반 스토리텔링)
4. CTA

### 아이디어 D. Products 페이지 - Neural Network Visualization

**컨셉**: AI 신경망 구조를 3D 노드/연결선으로 시각화. 데이터가 레이어를 통과하는 애니메이션.

**3D 효과**:
- 3D 공간에 노드(구체)와 연결선이 신경망 형태로 배치
- 데이터 패킷이 연결선을 따라 이동하는 펄스 애니메이션
- 노드 호버 시 활성화 + 정보 팝업
- 카메라가 스크롤에 따라 신경망 내부를 관통

**기술 핵심**:
- `InstancedMesh`로 수천 노드를 1 draw call로 렌더링
- 커스텀 셰이더로 연결선의 이동하는 하이라이트
- `UnrealBloomPass` 네온 글로우
- Raycaster로 노드 인터랙션

**난이도**: ★★★☆☆ (중)
**성능**: InstancedMesh 사용으로 가벼움. Bloom이 가장 큰 비용
**와우팩터**: ★★★★☆

**페이지 구성**:
1. 히어로: 거대한 3D 신경망 시각화
2. NexFlow 제품 상세 (신경망이 배경에서 작동)
3. NexInsight 제품 상세
4. 기술 비교표 + CTA

### 아이디어 E. Explore 페이지 - Infinite Tunnel / Wormhole

**컨셉**: 데이터 파이프라인 내부를 관통하는 워프 터널 효과

**3D 효과**:
- 네온 라인이 빛나는 터널 내부를 카메라가 관통
- 마우스로 터널 형태가 변형
- 스크롤로 속도 조절
- 별빛/파티클이 지나가는 효과

**기술 핵심**:
- `CatmullRomCurve3` + `TubeGeometry`
- 텍스처 UV 오프셋 스크롤로 무한 이동 착시
- 마우스로 curve.points 실시간 변경
- 커스텀 셰이더로 네온 라인 효과

**난이도**: ★★☆☆☆ (중하)
**성능**: 매우 가벼움. 단일 TubeGeometry + 텍스처 스크롤
**와우팩터**: ★★★☆☆

**페이지 구성**:
1. 히어로: 전체 화면 터널 진입 효과
2. 데이터 파이프라인 설명
3. 프로세스 단계별 소개
4. CTA

### 아이디어 F. Vision 페이지 - Gravitational Lensing / Black Hole

**컨셉**: AI의 "특이점"을 블랙홀 중력 렌즈 효과로 표현. 극강의 비주얼 임팩트.

**3D 효과**:
- 화면 중앙의 블랙홀이 주변 공간을 왜곡
- 마우스 위치가 왜곡 중심점
- 광원 디스크(accretion disk) 회전 효과
- 주변 파티클이 블랙홀로 빨려들어가는 애니메이션

**기술 핵심**:
- 풀스크린 커스텀 GLSL fragment shader
- UV 좌표를 중심점 거리 기반으로 비선형 왜곡
- `WebGLRenderTarget`으로 배경을 텍스처로 렌더링 후 왜곡
- `EffectComposer` + `ShaderPass` 후처리

**난이도**: ★★★★★ (상)
**성능**: 풀스크린 셰이더라 해상도에 비례. DPR 1.0 제한 필요
**와우팩터**: ★★★★★+

**페이지 구성**:
1. 히어로: 블랙홀 효과 + "The Singularity of AI" 텍스트
2. 비전/미래 전략
3. R&D 로드맵
4. 투자/파트너 CTA

## 추천 조합

### 1순위 추천: B + C (Globe + GPGPU Particles)

**이유**:
- **3D Globe (About)**: 포트폴리오에서 가장 범용적이고 인상적. Stripe 수준의 구현이면 면접관들이 감탄함. 추후 개인 사이트에서 "글로벌 프리랜서" 느낌으로 재활용 가능.
- **GPGPU Particles (Demo)**: 기술력 어필 최강. 10만 파티클이 텍스트/형태를 오가는 건 보는 사람 입이 벌어짐. GPGPU 경험은 시니어 프론트엔드 채용에서도 차별화 포인트.
- 메인의 파티클 구체 → GPGPU 파티클 모핑 → 인터랙티브 글로브로 3D 스킬 범위를 보여줌.

### 2순위 추천: A + F (Morphing Blob + Black Hole)

**이유**:
- **Morphing Blob (Technology)**: 가장 실용적. 구현 시간 대비 비주얼 효과 최고. 커스텀 셰이더 기초를 보여줌.
- **Black Hole (Vision)**: 압도적 비주얼. "이걸 웹에서 돌린다고?" 반응 보장. 다만 커스텀 GLSL 난이도 높음.
- 난이도 대비 극강의 임팩트 조합.

### 3순위 추천: D + B (Neural Network + Globe)

**이유**:
- AI 스타트업 컨셉에 가장 정합적. "이 사람이 AI 도메인을 이해하고 있구나" 느낌.
- 둘 다 중급 난이도라 구현 부담 적음.
- 다만 와우팩터가 위 두 조합보다는 살짝 낮음.

## 공통 성능 최적화 체크리스트

| 항목 | 설정 |
|------|------|
| `antialias` | `false` (후처리 사용 시) |
| `powerPreference` | `"high-performance"` |
| DPR 제한 | 데스크탑 1.0, 모바일 1.5 |
| 텍스처 해상도 | 2의 거듭제곱 (256, 512, 1024) |
| 비활성 탭 | 렌더 루프 중지 |
| 모바일 | 파티클 수/세그먼트 축소 |
| 모니터링 | stats.js로 FPS 확인 |
